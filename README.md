# LaTeX
LaTeX examples and templates

## Install

Check available packages
```
apt-cache search texlive
```

Packages
```
apt-get install texlive-base texlive-latex-recommended texlive texlive-latex-extra texlive-science texlive-lang-spanish
```

## Create and view PDF

Create PDF
```
pdflatex file.tex
```
View PDF
```
evince file.pdf
```
